## 개발 포인트

- [ ] AR 콘텐츠를 이용한 효과적인 주변 정보 표현
- [ ] 객채인식을 이용한 주변 견광의 정보 제공
- [ ] 가고자 하는 장소의 혼잡도 안내 (도로 혼잡상황 이용)
- [ ] 목적지 주변의 장소의 정보들을 다양한 테마에 맞게 제공
- [ ] 객채인식을 이용한 트렁크 내 물체 확인
